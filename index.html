Describe que hacen los siguientes comandos de git, escribe su descripcion al frente en una sola linea.

  1. git status: Este comando se usa para obtener el estado actual del repositorio
  2. git clone:Clona un repositorio en un nuevo directorio
  3. git pull:se emplea para extraer y descargar contenido desde un repositorio remoto y actualizar al instante el repositorio local para reflejar ese contenido
  4. git checkout:Cambiar ramas o restaurar archivos de árbol de trabajo
  5. git log:Mostrar registros de confirmación
  6. git branch:Listar, crear o eliminar ramas
  7. git add: Agrega el contenido del archivo al índice
  8. git commit:Registrar cambios en el repositorio
  9. git push:se usa para cargar contenido del repositorio local a un repositorio remoto
  10. git merge:se utiliza para fusionar uno o más ramas dentro de la rama que tienes activa
